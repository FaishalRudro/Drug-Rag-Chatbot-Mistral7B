# Drug-Rag-Chatbot-Mistral7B
A Retrieval-Augmented Generation (RAG) system for drug info. Handles queries on uses, dosages, side effects, warnings, and interactions. Supports Q&amp;A like "Abilify" or "Phentermine" info. Dataset scraped from Drugs.com. Provides context-aware, token-budgeted responses.
Dataset is available on request.
The demo of the project, can be found at (https://huggingface.co/spaces/RudroBoss/drug-rag-chatbot), as it's a free deployment platfrom, we had to change to demo model to gemma 2b.
